Home / Admin / Tambah Soal Quiz

➕ Tambah Soal Quiz

Tambah pertanyaan baru ke dalam quiz yang sudah ada

💡 Pilih quiz dari daftar. Nilai yang dikirim adalah Quiz ID dari tabel `quizzes`.
💡 Gunakan ini jika dropdown kosong. Quiz ID bisa dicek via: SELECT id, title FROM quizzes;

💡 Tips:

  • Pastikan quiz sudah dibuat di bagian Manage Content sebelum menambah soal
  • Untuk tipe Pilihan Ganda, isi semua opsi A-D dan pilih kunci jawaban yang benar
  • Poin default adalah 10, bisa diubah sesuai tingkat kesulitan soal
  • Urutan menentukan posisi soal saat ditampilkan ke user (1 = pertama)
  • PENTING: Gunakan Quiz ID dari tabel quizzes, BUKAN meeting_id. Cek via: SELECT id, title FROM quizzes;
